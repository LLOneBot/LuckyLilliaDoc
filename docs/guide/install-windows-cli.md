# Windows CLI 版本

## 版本特性

LLBot CLI 是命令行版本，适合熟悉命令行操作的用户和服务器部署场景。

### 主要特点

- ⚡ **轻量级** - 无图形界面，资源占用更少
- 🖥️ **命令行操作** - 通过命令行参数和配置文件控制
- 🔧 **灵活配置** - 支持更多高级配置选项
- 📦 **便于部署** - 适合服务器环境和自动化部署
- 🔄 **后台运行** - 可以作为后台服务运行
- 📝 **脚本友好** - 易于集成到自动化脚本中

### 适用场景

- 熟悉命令行操作的开发者
- 需要在服务器上部署的场景
- 希望最小化资源占用的用户
- 需要自动化部署和管理的场景

## 系统要求

- **操作系统**: Windows 系统（无特殊版本要求）
- **QQ 版本**: 64 位的 NTQQ（[官方下载](https://im.qq.com)）
- **注意**: 必须使用原版 QQ，不要安装任何插件

## 安装步骤

### 1. 下载安装包

前往 [GitHub Release 页面](https://github.com/LLOneBot/LuckyLilliaBot/releases) 下载最新版本的 `LLBot-CLI-win-x64.zip`

### 2. 解压文件

将下载的 zip 文件解压到任意目录，建议选择一个固定的位置（如 `C:\LLBot`）

### 3. 启动程序

打开命令提示符（CMD）或 PowerShell，进入解压目录，运行：

```bash
llbot.exe
```

### 4. QQ 登录

程序会自动启动 QQ，使用你的 QQ 账号登录即可

## 高级用法

### 命令行参数

CLI 版本支持多种命令行参数来自定义运行方式：

```bash
# 指定配置文件
llbot.exe --config=config.json

# 指定日志级别
llbot.exe --log-level=debug

# 更多参数请查看帮助
llbot.exe --help
```

### 配置文件

在程序目录下创建 `config.json` 文件来配置各项参数，详见 [配置指南](./config.md)

### 后台运行

可以使用 Windows 任务计划程序或第三方工具（如 NSSM）将 LLBot 设置为后台服务运行。

## 使用提示

- 建议创建一个批处理文件（.bat）来快速启动 LLBot
- 可以通过环境变量来配置一些参数
- 日志文件默认保存在程序目录的 `logs` 文件夹中

## 下一步

安装完成后，请查看 [配置指南](./config.md) 了解如何配置 LLBot 对接你的机器人框架。

## 常见问题

### 如何查看运行日志？

日志文件保存在 `logs` 目录下，可以使用文本编辑器查看。

### 如何停止程序？

在命令行窗口中按 `Ctrl+C` 即可停止程序。

### 可以在没有图形界面的 Windows Server 上运行吗？

可以，CLI 版本不需要图形界面，非常适合服务器环境。
