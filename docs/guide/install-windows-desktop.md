# Windows Desktop 版本

## 版本特性

LLBot Desktop 是带图形界面的 Windows 版本，专为新手用户设计。

### 主要特点

- 🎨 **图形化界面** - 无需命令行操作，所有功能都可以通过界面完成
- 🚀 **一键启动** - 双击即可启动，自动拉起 QQ
- 🔌 **框架对接** - 支持一键对接 Koishi、NoneBot 等主流机器人框架
- ⚙️ **可视化配置** - 通过界面直接配置各项参数，无需手动编辑配置文件
- 📊 **实时监控** - 可视化查看运行状态和日志信息
- 🔄 **自动更新** - 内置更新检查功能

### 适用场景

- 初次接触 LLBot 的新手用户
- 不熟悉命令行操作的用户
- 需要频繁调整配置的开发者
- 希望快速上手的个人用户

## 系统要求

- **操作系统**: Windows Server 2012 / Windows 10 及以上
- **QQ 版本**: 64 位的 NTQQ（[官方下载](https://im.qq.com)）
- **注意**: 必须使用原版 QQ，不要安装任何插件

## 安装步骤

### 1. 下载安装包

前往 [GitHub Release 页面](https://github.com/LLOneBot/LuckyLilliaBot/releases) 下载最新版本的 `LLBot-Desktop-win-x64.zip`

### 2. 解压文件

将下载的 zip 文件解压到任意目录，建议选择一个固定的位置（如 `C:\LLBot`）

### 3. 启动程序

双击 `llbot.exe` 文件，程序会自动启动

### 4. 启动 QQ

在 LLBot 界面中点击"启动"按钮，QQ 会自动启动

### 5. 登录 QQ

使用你的 QQ 账号登录，登录成功后 LLBot 会自动连接

## 使用提示

- 首次启动时，建议先查看设置界面，了解各项配置选项
- 可以在界面中直接配置 OneBot 协议的端口和其他参数
- 日志窗口会实时显示运行状态，遇到问题时可以查看日志排查

## 下一步

安装完成后，请查看 [配置指南](./config.md) 了解如何配置 LLBot 对接你的机器人框架。

## 常见问题

### 启动后 QQ 没有自动打开？

检查是否已经安装了 64 位的 NTQQ，并确保 QQ 没有在后台运行。

### 界面显示连接失败？

确保 QQ 已经成功登录，并检查防火墙是否拦截了 LLBot。

### 如何更新到新版本？

下载新版本后，直接覆盖原有文件即可，配置文件会自动保留。
