[真寻Bot](https://zhenxun-org.github.io/zhenxun_bot/) 是基于 NoneBot2 的功能丰富的 QQ 机器人。

<Callout type="info">
如果你使用 LLBot Desktop 版，可以在「对接框架」页面一键安装真寻Bot，无需手动操作。
</Callout>

### 前置依赖

- Python 3.11+
- [uv](https://github.com/astral-sh/uv)

<Steps>
<Step>
### 下载源码

```bash
git clone https://github.com/zhenxun-org/zhenxun_bot.git
cd zhenxun_bot
```

或下载 ZIP：https://github.com/zhenxun-org/zhenxun_bot/archive/refs/heads/main.zip
</Step>

<Step>
### 安装依赖

```bash
# 安装依赖
uv sync

```
</Step>

<Step>
### 启动

```bash
uv run zx
```
</Step>

<Step>
### 打开浏览器进行配置
根据提示打开`http://127.0.0.1:8080/#/configure`，会进入以下页面：

![_](/docs/asset/img/configuration/zhenxun-install-configure.png)

</Step>

<Step>
### 配置

* 在`超级用户ID`中填入你的QQ号或其他平台的ID。
* 如果你有其他数据库地址，可以在`数据库地址`中填入，否则保持默认即可。
* 登录用户名和登录密码是真寻前端的登录密码，必须填写，否则无法登录。
* 运行地址和运行端口（除非端口占用）一般默认即可。
</Step>

<Step>
### 完成配置
配置完成后点击 **提交** 出现弹框（直接选择确定）：
![_](/docs/asset/img/configuration/zhenxun-install-configure2.png)

选择 **确定** 之后页面将进行跳转到登录页：

同时后台将会开始重启并下载资源，等待资源下载完毕（总量大概在150mb左右）

</Step>
<Step>
至此真寻部分已经完全安装完成 

你可以接着进行配置LLBot
</Step>


<Step>
### 配置 LLBot

在 LLBot 中启用 OneBot11，添加反向 WS 连接：

```json5
{
  "type": "ws-reverse",
  "enable": true,
  "url": "ws://127.0.0.1:8080/onebot/v11/ws",  // 端口与 .env.dev 中的 PORT 一致
  "heartInterval": 60000,
  "token": "",
  "messageFormat": "array",
  "reportSelfMessage": false
}
```
</Step>


</Steps>
